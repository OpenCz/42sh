# 42sh
## Description
42sh is a simple Unix shell implementation that interprets and executes command lines. It supports basic shell features including command execution, piping, and redirection.

## Features
- Command execution with arguments
- Input/output redirection (`>`, `<`, `>>`)
- Piping between commands (`|`)
- Built-in commands (cd, exit, echo)
- Environment variable expansion
- Signal handling

## Installation
```bash
make
```

## Usage
```bash
./42sh
```
